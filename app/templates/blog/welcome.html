Welcome to your new website.

The first thing you need to do is create a superuser. To do this, run

in the root folder of the site.

After that, you will be able to log in to the admin panel and begin setting up and filling in the site.

To create the homepage, you can use two options:

  1. Create a folder app/templates/website/static_pages/home. Inside this folder, create files for each language (for example, en.html, de.html). To make the homepage appear, you need to create a file for at least the default language.
  2. In the admin panel, go to the "Site Content" section -> "Pages" and create a page with the URL "home". Note that data from the database takes priority over files on disk!
